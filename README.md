## Installation ğŸ“¥
1. Clone the repository:
   ```bash
   git clone https://github.com/0x0desha74/Talabat.API.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Talabat.API
   ```
3. Restore dependencies:
   ```bash
   dotnet restore
   ```
4. Update the database:
   ```bash
   dotnet ef database update
   ```
5. Run the application:
   ```bash
   dotnet run
   ```
